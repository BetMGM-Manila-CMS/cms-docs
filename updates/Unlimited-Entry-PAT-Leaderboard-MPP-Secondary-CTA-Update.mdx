> Hello everyone,

> For unlimited entry, PAT Leaderboard, the MPP secondary CTA has always been either See Standings for BetMGM or SEE STANDINGS for all other labels except BetMGM, for Leaderboards and Check My Entries for BetMGM or CHECK MY ENTRIES for all other labels except BetMGM, for Sweepstakes, directing to !!M2.Promo/offers/grouped?filter=cp, so that players are directed to the Filtered, Casino PAT Promotions Hub Page to search for the specific PAT Leaderboard, Promotions Hub Tile.

> However, Alex visited our Canadian office and received feedback from Paul, sharing with us during last month’s CMS/OPS Monthly Connect that we should optimize our user journey with less steps, to become See Rankings for BetMGM or SEE RANKINGS for all other labels except BetMGM, regardless of Leaderboard or Sweepstakes, directing to !!M2.Promo/lb/PAT ID #, but with the PAT ID # replaced with the actual number instead, for example, 123456 for !!M2.Promo/lb/123456, so that players are immediately directed to the PAT Leaderboard Page, specifically the Ranking Tab in which the URL will automatically prepopulate and end with …/promo/lb/PAT ID #/ranking if a player has successfully opted-in and made an eligible or qualifying wager, or specifically the Overview Tab in which the URL will automatically prepopulate and end with …/promo/lb/PAT ID #/overview if a player has not made an eligible or qualifying wager yet.

> Please note that this update is for single Promotional Period, unlimited entry, PAT Leaderboard, which is what is usually run most of the time, and the Parent Final PAT ID will not reproduce Children Final PAT IDs, since the promotion will start and end just once. 

> However, this update is not for multiple Promotional Periods, unlimited entry, PAT Leaderboard, which is not what is usually run most of the time. For example, for this Promotion, the Parent Final PAT ID reproduced three (3) Children Final PAT IDs, since this Promotion will start and end more than just once, so the MPP secondary CTA can still be See Rankings for BetMGM or SEE RANKINGS for all other labels except BetMGM, regardless of Leaderboard or Sweepstakes, but directing back to !!M2.Promo/offers/grouped?filter=cp instead:
