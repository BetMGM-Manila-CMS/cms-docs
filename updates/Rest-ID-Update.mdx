Hi all,

Sharing an update from the Ops team as it relates to building + pushing rest IDs to optimove.

Under ‘OptimoveIntegration’ and OptimoveInstance - where we’d normally tick ‘RoarDigital’, going forward we will be using the two highlighted sections below; BETMGMCA for anything in Canada (BetMGM Ontario, WOF Ontario and Alberta labels as they launch) and BETMGMUS for everything else – Borgata, WOF, PartyCasino, etc. in the US. Please tick the applicable box before pushing to optimove.

Please use this setup going forward. Any previously created rest IDs with RoarDigital ticked can stay as is, no need to make updates.
