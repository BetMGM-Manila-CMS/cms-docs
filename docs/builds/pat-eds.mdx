# PAT and EDS

PAT and EDS have different use cases but most of the components' fields have some similarities.

At the same time with other use cases often when it is being used with `Engagement Tickets` there are many fields that are specific with those type of tickets. Please refer to the guides under the PAT and EDS section.

## Editing your PAT or EDS

In contrast with other components, the PAT and EDS is provided by the Ops team. We can either get the respective PAT or EDS from the `Updates` or `Project Details` tab under the `CRM Ops` Section of your Workfront ticket.

## Terms

### KeyTermsAndConditions

The value for this field will be from the `Promo Hub Key Terms > Short Summary` from the copy.

<Admonition type="caution" title="Note">
  Enclose this content with the \<p>\</p> tag
</Admonition>

### Full Terms and Conditions

<Admonition type="note" title="Note">
  If the Terms have mentioned for games that **must be wagered**, the tiles for these games will also be included at the top of the Terms and Conditions.

  ![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887485/builds/pat-eds/p7jgv9fvvbwttugqdiww.png)
</Admonition>

<Admonition type="tip" title="Tip">
  The content for the Terms and Codition will be the same for the content from `MPP` or `Rest ID`, so feel free to just copy and paste the content from those components. For a reference please refer to this [guide](./documents#terms-and-conditions).
</Admonition>

<Admonition type="note" title="Note">
  If Promotion is SHARED Leaderboard on Multiple states (e.g., NJ, PA, MI, ON, WV):
  Do not add game tiles to PAT T\&Cs because the game links vary between states.

  If Promotion is SHARED Leaderboard on same state (e.g., BC NJ, MC NJ, PC NJ):
  You can add the game tile, but double-check that the game links are identical.
</Admonition>

### Share PAT CTA Configuration

If the promotion is a Shared PAT there's an additional step we need to take.

Firstly `!!M2.CasinoHome/c/newgames` will be the default CTA if it is a Shared Promotion.

Click on the dropdown arrow to show the advanced view of the CTA configuration.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1756883652/updates/shared-pat-play-now-config/sharepatctaconfig_image1_wartbq.png)

Search for the label that is included in your promotion using the dropdown and search for its corresponding game link, and paste it in the URL field.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1756884165/updates/shared-pat-play-now-config/sharepatctaconfig_image2_h8k2db.png)

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1756884618/updates/shared-pat-play-now-config/sharepatctaconfig_image3_m20otn.png)

If the PAT is shared across multiple states, you can add more rows in the label simply by saving the PAT; it will automatically add another row to the Label.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1756881831/updates/shared-pat-play-now-config/sharepatctaconfig_headerimg_umst2m.png)

## Rewards and Results options

This field is a dropdown where we need to select the appropriate folder depending on promotion type of your ticket.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887484/builds/pat-eds/uc8vmtj0xbnybnzvbz6y.png)

<Admonition type="tip" title="Tip">
  The appropriate folder will be under the DetailPagesContent folder inside the state/brand folder. The state/brand should be the same as stated in your ticket. Here is the path to DetailPagesContent

  Content > Vanilla.Mobile > State/Brand > Promo > PromoHub > DetailPagesContent
</Admonition>

<Admonition type="note" title="Note">
  There are always two final folder destination to select, those are `Default` and `Default-US`, only pick `Default` folder if `Default-US` is not present.
</Admonition>
