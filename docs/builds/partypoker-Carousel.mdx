# partypoker Carousel

![](https://res.cloudinary.com/dlfu36fiw/image/upload/c_scale,w_0.5/builds/poker-carousel/partypoker-caro_ng7ghg.png "default")

For partypoker's carousel, it is structured differently, the layout is different and in Sitecore it is also created on a separate path.

<Admonition type="note" title="Sitecore Path">
  Content > Vanilla.Mobile > m2.nj.partypoker.com > MobilePoker-v1.0 > index > slides
</Admonition>
