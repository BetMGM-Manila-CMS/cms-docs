# Casino Carousel

The Casino Carousel is the main attraction of the Casino Page.
It shows all the latest promos, Public and Targetted, to the customers. In this section, we will provide the necessary steps in building the Casino Carousel.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1714980640/builds/carousel/casino-carousel-sample_ism2z4.png "default")

## Creating The Carousel

To create your Carousel, navigate to it's directory:

<Admonition type="info" title="Sitecore Path">
  Content > Vanilla.Mobile > State/Brand > MobileCasino-v1.0 > Teasers > Repository
</Admonition>

Duplicate one of the Carousel object, preferably the template.

<img src="https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887462/builds/carousel/q6sxdixd81nm41rocaeq.png" className="mb-3" />

Name the new Carousel object with the Ticket name.

<img src="https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887456/builds/carousel/dwomnqhxhnqi7dizfqub.png" />

### Desktop and Mobile Carousels?

Most of the time we create 2 version of the carousel, one for mobile one for desktop. This is based on the supplied asset from the designers. In this case we need to check for 2 assets: `605x328` and `Casino Carousel-1920x560`.

If there is no provided `605x328` we need to create a desktop and mobile carousel, but if there is, we only need to create a single carousel.

<Admonition type="note" title="Naming Carousels">
  For desktop and mobile carousels, simply suffix a `-Desktop` and `-Mobile` to label the appropriate carousels
</Admonition>

## Pushing The Carousel

To push your newly created Carousel, navigate to this directory:

<Admonition type="info" title="Sitecore Path">
  Content > Vanilla.Mobile > State/Brand > MobileCasino-v1.0 > LMTCategories > featured/lobbyhome
</Admonition>

<img src="https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887456/builds/carousel/zmsaop1y1buoaj1wixa3.png" className="mb-3" />

Find your Carousels in the left box then select and click the right arrow button. Do this for both Desktop and Mobile carousels.

## Previewing The Carousel

To preview your Carousel go to your `Product and State Website` and then add your `filter`.

See example below:

```
https://casino.nj.betmgm.com/en/games?qa=manila
```
