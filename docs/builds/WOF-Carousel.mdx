# WOF Carousel

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1715236709/builds/carousel/wof-carousel_hrmjjo.png "header-img")

For Wheel Of Fortune Casino, the difference is that the path to create a Carousel Slide is different and there is no need to *push* each Carousel Item.

<Admonition type="note" title="Sitecore Path">
  Content > Vanilla.Mobile > m2.wheeloffortunecasino.com > MobileCasino-v1.0 > OptimizedTeasers > TeaserItems > HomeF
</Admonition>

## Creating Your Carousel

We can just follow the usual steps for creating the usual casino carousel from the [parent guide](./casino-carousel), and always create a new carousel slide under the `RECENTLY BUILT BY MANILA` marker.

### Required Fields

Compared to the usual casino carousel build, the fields are mostly similar with just the order quite change and some changes to note.

<Admonition type="note" title="Carousel Limit">
  The WOF Carousel is limited to 6 slides. Therefore, if there are active slides in the 7th position and so on, they won't be displayed on the Carousel. The order of each carousel slide will depend on the order on Sitecore.
</Admonition>
