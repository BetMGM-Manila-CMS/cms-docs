`PCText` contains the text which is usually is displayed at the bottom of a page, but it also contains the page description when it is searched on any browser.

## Pushing the SEO Box

To ensure the SEOBox is applied correctly, it is required to be pushed to the appropriate `LMTCategories` item. If it has, no further action is needed. If not, follow these steps: 

1. Inspect your page and go to the `Applications` tab. 
2. Click the website's URL under the Local Storage category.
3. Search for `categoryCoreLookUp_en_instantCasino_casino`, click the result, and check its value.
4. Look for the `lsCategoriesMap` under `value`, an array showing the LMTCategories names that need updating. This array is ordered the same way page links are displayed on the website. This name coresponds to the `LMTCategories` item that you would update on Sitecore.
