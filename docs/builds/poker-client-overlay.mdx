## Building in Sitecore

There are only two sections you need to update: the **DownloadClientPromotionData** and the **Filter** sections.

### Filter

Use your QA filter, e.g., `QueryString.Get('qa') = 'manila'`

To preview the client overlay, go to the client overlay preview URL, adjusting it based on the state/label of your ticket and the QA filter you used.

For example, if you're working on a ticket for Borgata NJ and you used the QA filter `Request.Query MATCHES 'manila'`, then the client overlay preview URL will be: `https://www.borgataonline.com/en/mobileportal/clientplppreview?manila`
