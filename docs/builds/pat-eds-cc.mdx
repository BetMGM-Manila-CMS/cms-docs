# Click Card

<Admonition type="info" title="Copy and assets needed">
  * **Card Images** - Look for the `CRM` folder in the ticket's creatives.
</Admonition>

Compared to the other engagement promos, Click Card's PAT does not deviate much to the default PAT. The only additional field is the [Cards](#cards) section, where we will place the 2 cards asset for this promotion.

## Cards

Use the same image for both fields of `CardImage` and `FreeCardImage`.
