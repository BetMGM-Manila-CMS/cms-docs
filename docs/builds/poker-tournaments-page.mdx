# Tournament Page

<Admonition type="info" title="Sitecore Path">
  sitecore > content > Vanilla.Mobile > state/brand > MobilePoker-v1.0 > Tournaments
</Admonition>

In this page, you'll learn how to build a Tournament Page for Poker, which consists of two subcomponents: the Tournament Page itself, and the Side Navigation.

If Casino has MPP, Poker has Tournament Page or Tourney Page. Compare to MPP, Tournament Page requires many components in the copy such as **PROMO HUB KEY TERMS**, **Poker Tile**, **PROMO LANDING PAGE**, and **Terms & Conditions**.

## Start Building...

### Step 1: Duplicate File

It's always better to duplicate the file with same promo you'll build. Like if it's Leaderboard, look for file that's build for Leaderboad; Champoinship for Champoinship; and the like.

![This is](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887577/builds/tourney-page/drb7dk589mvcorz7sgjc.png)
*Right-click your desired file, choose Duplicate, then rename it with your URL*

### Step 2: Update SEO

Fill the following field using the **PROMO HUB KEY TERMS** from the copy.

* Title ➡ H1
* Description ➡ Subtitle

![SEO Preview](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887513/builds/tourney-page/te7m43qzxluw1g0uvsxc.png)
*Make sure that there are no double space, and space after the sentence/phrase*

<Admonition type="note" title="Note">
  for `partypoker` add this part after the H1 Title. ![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1720429282/builds/tourney-page/Screenshot_7_labnwr.jpg)
</Admonition>

### Step 3: Update Synopsis

Fill the followng fields using the **Poker Tile** from the copy.

* SypnosisImage ➡ Poker Tiles-640x320 Asset
* SypnosisTitle ➡ H1
* SypnosisDescription ➡ Subtitle

![Synopsis Preview](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887514/builds/tourney-page/tqrfbhkeghgw8ac3onfd.png)

### Step 4: Update MainContent1

This Content should follow this format:

```html
<h1><strong>{Promo Landing Page Main Image}</strong> </h1>
<h2><strong>{Promo Landing Page Title/Subtitle}</strong><h2>
<h2 style="color: #(State Hex Color);"> <strong>Promo Date</strong>\</h2>
<p>Sentence case. Promotional description</p>
<p>Most of the time this contains Table, follow Table formatting on the link provided below.</p>
<p><em>Please make sure to read our full Terms and Conditions before participating in this promotion.</em></p>
<p><a class="btn" style="background: #{State Hex Color); color: #ffffff;" href="{SmartLink}">{CTA}</a></p>
```

Fill this content using the following:

### Step 5: Update Terms

The Poker Tournament Page's Terms & Condition has an additional line compare to other builds. Copy the code below and add this before your Ticket's Terms & Conditions.

```html
<h2>${dic:4BBA6E66-85C6-4D2F-ABC0-5BA4C06D20CE/TermsandConditions}</h2>
```

## Filter & Publishing Dates

### Step 8: Filter

Both Tourney Page and Side-Nav must have filter.

```
QueryString.Get('qa') = 'manila'
```

![Filter](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887515/builds/tourney-page/euylgiqmrounim8ukfis.png)

### Step 9: Publishing Date

Both Tourney Page and Side-Nav must have the same publishing date. If it's a new build, the publishing date should be empty and the Never Publish must be ▢ unticked.

![Publishing Date](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887516/builds/tourney-page/e6ohberwn0fhulvbqtrl.png)
