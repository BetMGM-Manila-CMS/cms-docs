# Tournament Page

:::info[SiteCore Path]
sitecore > content > Vanilla.Mobile > state/brand > MobilePoker-v1.0 > Tournaments
:::

If Casino has Marketing Promotional Page (MPP), Poker has Tournament Page. Compare to MPP, Tournament Page or Tourney Page requires many components in the copy such as **PROMO HUB KEY TERMS**, **Poker Tile**, **PROMO LANDING PAGE**, and **Terms & Condition**.
