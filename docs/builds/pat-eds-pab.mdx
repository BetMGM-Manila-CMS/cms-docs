# Pick A Box

<Admonition type="info" title="Copy and assets needed">
  * **Prize Tool** - You'll find this in the Documents section of your ticket's Workfront.
  * **Box** - Look for the `PAB Assets` folder in the ticket's creatives.
</Admonition>

<Admonition type="info" title="Copy and assets needed">
  The Promotion type dictates the *Box* asset to be used, as indicated by the ticket name. PAB promotions involve selecting a **Box**, **Present**, **Card**, and so forth, each corresponding to a designated asset.
</Admonition>

## Offer

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887555/builds/pat-eds-pab/kwnbmkjzw3kgqk4yykpz.png)

1. Place an empty `<p></p>` tag for the value of the description.
2. The KeyTermsAndConditions will take the value of the `Short Summary` in your copy document enclosed in `<p></p>`
3. For the Teaser Image place the `Promohub Tiles-800x360` image.
4. Place `"Reveal a mystery prize"` for the TeaserClass.

## BoxSection

In this section is where the contents from the [Prize Tool](./documents#prize-tool) document will be placed. Each row will contain a prize title, an image that is assigned to that specific reward, and a description which contains a message and a CTA to direct the user to either claim or use the reward that has been granted to them.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887554/builds/pat-eds-pab/bxvgkazkgt3nyz44y4jz.png)

1. **Title** - Place the the contents from the `PRIZE` column from the `Prize Tool` document in the same order to the BoxSection's `Title` Rows.
2. **Image** - For the images use the `240x240` image where it *doesn't show the prize* ([see note below](#boxImageTip)), and place it on all the rows only if there is only a single image provided, but for multiple different images place it the same order as in the creatives folder order.

<Admonition type="note" title="Unopened Prize Image">
  <div id="boxImageTip" />

  The images may vary per promotion, for example for Pick a Box the image to use is the unopened box image, for Pick a Card its the facing down card image, etc.
</Admonition>

<Admonition type="tip" title="Note">
  Use the image url by clicking the `Open` link of the image on the Media Library. And copy the link on the URL and paste on the image column of the `BoxSection` table.
</Admonition>

1. **Description** - The `Description` column's value will be the value from the `PRIZE DESCRIPTION` column from the `Prize Tool` document, but it should be inserted in the HTML format shown below, just replace the appropriate value to be the same from your `Prize Tool` document.

   <br />

```html
<div className="overlay-header">
<h1 className="mb-1 mt-0">Congratulations!</h1>
<h2 className="mb-2 mt-1">You've won: $1000 Online Open Hybrid Tournament Ticket</h2>
<p className="mb-2">Register in our tournament lobby for the Online Open Hybrid Tournament</p>
<p className="mb-2">Tickets expire after 90 days. T&amp;C's apply.</p>
<div className="mt-2">
<a rel="noopener noreferrer" className="btn btn-primary" href="{Reward CTA URL}">{Reward CTA Text}</a>
</div>
</div>
```

<Admonition type="note" title="CTA">
  See the [CTA guide](https://coralracing-my.sharepoint.com/:x:/g/personal/krisha_manalo_entaingroup_com/EfvRGdc4sN5MiwcmKjBu55wBGvjTEIDYp-1XEU-iLJkGXA) to use the correct CTA for each description.
</Admonition>

## Content Section

<Admonition type="tip" title="Title Fields">
  For the Title Fields for `T & C Header` and `How It Works Title`, the values here doesn't need to be changed.
</Admonition>
