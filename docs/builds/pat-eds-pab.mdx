# Pick A Box

:::info[Copy and assets needed]

- **Prize Tool** - You'll find this in the Documents section of your ticket's Workfront.
- **Box** - Look for the `PAB Assets` folder in the ticket's creatives.

:::note

The Promotion type dictates the *Box* asset to be used, as indicated by the ticket name. PAB promotions involve selecting a **Box**, **Present**, **Card**, and so forth, each corresponding to a designated asset. 

:::

## Offer

<img className="mt-3 mb-5" src={offer} />

1. Place an empty `<p></p>` tag for the value of the description.
2. The KeyTermsAndConditions will take the value of the `Short Summary` in your copy document enclosed in `<p></p>`
3. For the Teaser Image place the `Promohub Tiles-800x360` image.
4. Place `"Reveal a mystery prize"` for the TeaserClass.

## BoxSection

In this section is where the contents from the [Prize Tool](./documents#prize-tool) document will be placed. Each row will contain a prize title, an image that is assigned to that specific reward, and a description which contains a message and a CTA to direct the user to either claim or use the reward that has been granted to them.

<img className="mt-3 mb-5" src={boxSection} />

1. **Title** - Place the the contents from the `PRIZE` column from the `Prize Tool` document in the same order to the BoxSection's `Title` Rows.


2. **Image** - For the images use the `240x240` image where it *doesn't show the prize* ([see note below](#boxImageTip)), and place it on all the rows only if there is only a single image provided, but for multiple different images place it the same order as in the creatives folder.

    <br />

    :::note[Unopened Prize Image]

    <div id="boxImageTip" />

    For box images its the unopened box image, for cards its the facing down card image, etc.

    ::: 

3. **Description** - The `Description` column's value will be the value from the `PRIZE DESCRIPTION` column from the `Prize Tool` document, but it should be inserted in the HTML format shown below, just replace the appropriate value to be the same from your `Prize Tool` document.
    
    <br />

    ```html
    <div class="overlay-header">
    <h1 class="mb-1 mt-0">Congratulations!</h1>
    <h2 class="mb-2 mt-1">You've won: $1000 Online Open Hybrid Tournament Ticket</h2>
    <p class="mb-2">Register in our tournament lobby for the Online Open Hybrid Tournament</p>
    <p class="mb-2">Tickets expire after 90 days. T&amp;C's apply.</p>
    <div class="mt-2">
    <a rel="noopener noreferrer" class="btn btn-primary" href="{Reward CTA URL}">{Reward CTA Text}</a>
    </div>
    </div>
    ```
    :::note[CTA]

    See the [CTA guide](./cta) to use the correct CTA for each description.

    :::

<img className="mt-3 mb-5" src={criteria} />

```html
    .ptb .spin-result-dialog .overlay-content {

    }

```

:::note
