# SHARED Leaderboard & Sweepstakes

SHARED Leaderboard/Sweepstakes PAT ID differ slightly than the regular PAT IDs since this type of promotion is shared between different Labels and States.

The main difference are the required inputs and assets in the `Overview` section and `How It Works` section.
