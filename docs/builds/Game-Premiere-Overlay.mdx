# Game Premiere Overlay

A Game Premiere Overlay is a component on the casino homepage of a product, that plays once after you login your account in BetMGM.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1746682222/builds/game-premiere-overlay/GamePremiere_Image1_nwgneb.png)

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1746683926/builds/game-premiere-overlay/GamePremiere_Image3_a4zg1i.png)

It is the same as the Video Banner which gets updated from time to time instead of creating a new component each promotion. To update the Game Premiere here is the Sitecore Path

<Admonition type="info" title="SiteCore Path">
  Content > Vanilla.Mobile > State/Brand > MobileCasino-v1.0 > GamePremiere > GamePremiereOverlay
</Admonition>

<Admonition type="info" title="Note">
  Item name must be kept as 'GamePremiereOverlay' in Sitecore because ODR and non-ODR are configured to follow the same path, and if item is renamed, overlay will not display for ODR.
</Admonition>

## Assets

The uploading of assets is the same as the one we do in the Netflix Video Banners. See this guide [here](https://betmgm-manila-cms-docs.netlify.app/docs/builds/assets-uploading/#uploading-mp4-file-for-video-banner "here").

### Steps

This section contains 3 steps that summarizes how the players can join and receive their rewards.

The copy has the `How It Works Tab` section which contains the value for each step. Place the description of each step accordingly.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1746685505/builds/game-premiere-overlay/GamePremiere_steps_ibxhfo.png)

### CTA

Place the Game Smartlink of your promotion in the `playnowCTA`.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1746685502/builds/game-premiere-overlay/GamePremiere_playnowCTA_yv2ysx.png)

### MoreInfo

Add the Terms and Conditions here.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1746685514/builds/game-premiere-overlay/GamePremiere_moreInfo-Terms_mxizu1.png)

Add the `See Rankings` or the Secondary CTA of your MPP and its Smartlink here.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1746685511/builds/game-premiere-overlay/GamePremiere_moreInfo-viewDetails_upkptf.png)

### SourceTypes

For the `sourceType` select the one that is appropriate for your promotion and input its PAT/EDS ID in the `sourceId`

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1746685508/builds/game-premiere-overlay/GamePremiere_sourceSettings_vvrurd.png)

<Admonition type="note" title="Note">
  If the promotion has no PAT/EDS ID like Bonus Drops/Spins you can input `Static` for the `sourceType` and leave the `sourceId` blank.
</Admonition>

### Filter and Publishing Date

<Admonition type="caution" title="Always Set to Never Publish">
  For the Manila Team make sure to always tick the Never Publish since the US team will be the one testing and double checking this components since this requires a targeted placeholder PAT/EDS ID.
</Admonition>
