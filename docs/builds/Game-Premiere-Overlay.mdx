# Game Premiere Overlay

A Game Premiere Overlay is a component on the casino homepage of a product, that plays once after you login your account in BetMGM.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1746682222/builds/game-premiere-overlay/GamePremiere_Image1_nwgneb.png)

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1746683926/builds/game-premiere-overlay/GamePremiere_Image3_a4zg1i.png)

It is the same as the Video Banner which gets updated from time to time instead of creating a new component each promotion. To update the Game Premiere here is the Sitecore Path

<Admonition type="info" title="SiteCore Path">
  Content > Vanilla.Mobile > State/Brand > MobileCasino-v1.0 > GamePremiere > GamePremiereOverlay
</Admonition>

<Admonition type="info" title="Note">
  Item name must be kept as 'GamePremiereOverlay' in Sitecore because ODR and non-ODR are configured to follow the same path, and if item is renamed, overlay will not display for ODR.
</Admonition>

## Assets

The uploading of assets is the same as the one we do in the Netflix Video Banners. See this guide [here](https://betmgm-manila-cms-docs.netlify.app/docs/builds/assets-uploading/#uploading-mp4-file-for-video-banner "here").
