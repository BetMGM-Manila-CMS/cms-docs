# CTA

This page serves as a guide for choosing the right CTA links and labels according on the kind of ticket you are managing. This feature is used by many components to smoothly reroute customers to the desired sites.

The CTAs uses smartlinks to redirect to specific pages without specifically typing the domain for the state and product. To see more information refer to this [guide](smartlinks).

:::tip

The Opt In CTA is an HTML element that has specific attributes, copy the element from the [section](#opt-in) below.

:::

## Opt In

The Opt In CTA is a button or link that allows the user to join a specific promotion, its HTML element consists of different attributes and where we also place the `PAT ID`. It is often used for the [MPP](#).

```html
<a 
	data-offer-id="{PAT ID}" 
	className="btn" 
	style="background-color:{Promo and Brand Color};" 
	data-offer-message-error="Technical Error" 
	data-offer-message-expired="Expired" 
	data-offer-message-invalid="Invalid" 
	data-offer-message-not-offered="Not Offered" 
	data-offer-message-offered="Opt In" 
	data-offer-message-opted-in="You opted in" 
	data-offer-message-opted-out="You opted out" 
	data-offer-type="{promos | eds}"
>
	Opt In
</a>
```
