# Component Scheduling

## Publishing

Scheduling components like Carousel, and ODR Banner to push them to live.

Publishing section is found at the bottom of the component page.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1718346392/publishing/image_2024_06_05T07_21_51_983Z_zkepsu.png)

You may also find the PUBLISH tab at the top, then click 'Change' (Globe Icon).

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1718346389/publishing/image_2024_06_05T07_22_08_587Z_n3ykbg.png)

Click on the Item tab, to set the schedule of the promo.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1718346385/publishing/image_2024_06_05T07_22_19_378Z_sgysei.png)![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1718346382/publishing/image_2024_06_05T07_22_29_668Z_wekj3b.png)![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1718346378/publishing/image_2024_06_05T07_22_49_979Z_zckx3b.png)![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1718346375/publishing/image_2024_06_05T07_23_03_285Z_sczisv.png)

Note: Time is set to 4:50. 'Publishable from' is set to starting date, then 'Publishable to' is set to end date + 1 day.
Note: Tick the check box 'Publishable' to set publishing, untick to 'unpublish'
