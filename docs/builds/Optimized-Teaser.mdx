# Optimized Teaser

The Optimized Teaser will be the new main attraction of the Casino Page in selected states replacing the Casino Carousels.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1730687630/builds/optimzied-teaser/headerimg_uwbiql.png)

## Creating Optimized Teaser

To create an Optimized Teaser, navigate first to its directory:

<Admonition type="info" title="Sitecore Path">
  Content > Vanilla.Mobile > State/Brand > MobileCasino-v1.0 > OptimizedTeasers > TeaserItems > Live\_Folder
</Admonition>

Duplicate one of the Optimized Teaser.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1730688076/builds/optimzied-teaser/duplication_uftnu4.png)

Then rename the duplicated item with the Ticket name.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1730688079/builds/optimzied-teaser/renaming_jplkz3.png)

### Parameters

Edit the `link-url` parameter into the CTA smartlink of your ticket.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1730688845/builds/optimzied-teaser/parameter_wrqarj.png)

### Filter

Add your QA filter just like in the Casino Carousel.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1730689243/builds/optimzied-teaser/filter_qcp8ys.png)

### Imagery

Fill the `Background Image` with the "ODR - Teaser BG Image - 900x450" media URL.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1730689389/builds/optimzied-teaser/backgroundimage_s4hk55.png)

## Previewing the Optimized Teaser

To preview your Carousel go to your `Product and State Website` and then add your `filter`.

See example below:

```
https://casino.nj.betmgm.com/en/games?qa=manila
```
