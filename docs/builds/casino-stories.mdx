# Stories

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887576/builds/stories/rzzxufc62enzm1lhdus5.png)

## Building the Story

Casino stories has two components, InstaContainer and StoryContent.

At the moment, only **Borgata NJ** has the Stories feature.

## StoryContent

<Admonition type="note" title="Sitecore Path">
  Content > Vanilla.Mobile > m2.borgataonline.com > MobileCasino-v1.0 > InstaStories > StoryContent

  ![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887573/builds/stories/qqfjr8m2hg9kaweb8iaz.png)
</Admonition>

Duplicate any component, and rename it to your ticket.

StoryContent has 2 child components, CSS and HTML.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887573/builds/stories/j2wm6uk4fmsk8iqpj1l4.png)

<Admonition type="tip" title="Tip">
  Naming convention for "Stories" are no spaces, and no special characters except underscore, '\_' .
</Admonition>

### CSS

Edit the `backgroundImage` parameter, fill the "1080x1920-Stories" media link.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887570/builds/stories/bpz1gqlrknqoeouo3m4d.png)

### HTML

Edit the 'HTML editor', set the same CTA for Carousel/ODR Banner.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887571/builds/stories/vcz471wgjz9ua54sqqic.png)

```html
<div className="casino-background-image-section">
<div className="img-promo-container">
<div className="imgbackground">
<div className="story-text">
<div><br></div>
</div>
<div>
<div className="optinBtn">
<a href="!!M2.Portal/mobileportal/promotions/casino/url" className="btn btn-primary more-info-btn">LEARN MORE</a>
</div>
</div>
</div>
```

## InstaContainer

<Admonition type="note" title="Sitecore Path">
  Content > Vanilla.Mobile > m2.borgataonline.com > MobileCasino-v1.0 > InstaStories > InstaContainer > InstaPromotions

  ![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887572/builds/stories/zr70msc5waltvadet3ae.png)
</Admonition>

Duplicate any component, and rename it the same as the StoryContent component. Also rename the child component as well.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887572/builds/stories/zvozt3ri0g3fsomrmkym.png)

### Parent

1. Fill the `StoryThumbnailImage` with the "Story - 500x500\_round" media URL.
2. In Stories section, select your child component and push it to selected.
3. Fill the `Title` with Promohub H1.
4. Fill the `Condition` with your QA filter.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887574/builds/stories/rtkhchocs80zv3gemzlj.png)
![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887508/builds/stories/eef9tixndypllpstjsfx.png)

### Child

In the MainStoryContent, find and select your newly created StoryContent.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887508/builds/stories/vz8zl9u5imvmlotqsobc.png)
