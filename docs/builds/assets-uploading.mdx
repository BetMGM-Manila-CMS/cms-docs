# Uploading Creatives

:::warning

Ensure that all files are below required [minimum size](./assets-introduction#exceeded-required-file-size) as the Audit team regularly checks files that are above the required file size and may report it.

:::

## Sitecore Path

The parent path for uploading assets is located on:

```
Media Library > Vanilla.Mobile > Product State Domain
```
After getting on your Product State Domain Check guide below for the specific folders for each product

| Product | Path |
| --- | --- |
| Casino | `Year > Month` |
| Poker | `Poker Folder > Year > Month` |
| Sports | `Sports Folder > Year > Month` |
