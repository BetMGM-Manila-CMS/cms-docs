# Desktop PLP

**Desktop PLP** is a carousel-like feature but takes up the whole screen of a desktop device.

<img src="https://res.cloudinary.com/dlfu36fiw/image/upload/v1714984968/builds/plp/desktop/sample_gwdc6g.png" className="mx-auto" />

## Building desktop PLP

<Admonition type="info" title="Sitecore Path">
Content > Vanilla.desktop > brand/state > desktopPoker-v1.0 > Regional > Global > desktopPLP > desktopPLP-LIVE
</Admonition>

Go to the above Sitecore Path. Then do the following:

### Step 1. Duplicate File

1. Right-click the file you want to duplicate.
2. Choose Copy To
3. Remove the URL of your choosen File
4. Click Copy
5. Rename the file to your ticket's name

### Step 3: Filter & Publishing Date

1. Insert your QA Filter under **Condition**

```html
QueryString.Get('qa') = 'manila'
```

1. Clear Publishing Date (for new builds)

### Step 4: Screenshot your PLP

Go to your sites plp preview page with your qa filter name, then look for your output.

```html
https://poker.nj.betmgm.com/en/clientplp?qa=manila
```
