# Coin Flip

One of the Casino Engagements is the Coin Flip. In this game, the player flips a coin and predicts which side it will land on.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887461/builds/coin-toss/hikmlsdaiqcyfm2rajae.png)

To build a Coin Flip Page, you'll need to have the following sitecore files:

1. Layout
2. PAT

## Layout

Here, you'll edit the layout of the Coin Flip. Follow the Path below to start building.

<Admonition type="info" title="SiteCore Path">
  content > Vanilla.Mobile > brand/state > Promo > Engagement > c
</Admonition>

### Step 1: Duplicate File

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887529/builds/coin-toss/bbqp19xylmkjwckpmpa8.jpg)

1. Right-click the file you want to duplicate.
2. Choose Copy To
3. Remove the URL of your choosen File
4. Click Copy
5. Rename the file to your URL

### Step 2: Edit **assetsandcontent**

<ol>
  <li>
    Click the drop down icon (►) of your file, then go to assetsandcontent
  </li>

  <li>
    Edit the **Toss** Section
  </li>

  <ul>
    <li>PreBetText</li>
    <li className="list-none">Just edit the Gold or Purple part, depending on the color of your Coin *(as seen in the assets)*.</li>

    <li className="list-none">
      ```html
      <span class="txt-white" style="font-size: var(--site-spacer-l)">
        It's all in your hands
      </span>

      <span>Make a bet! is it going to be</span>
      <span>Gold or Purple?</span>
      ```
    </li>

    <li>TossConfirmationText</li>

    <li className="list-none">
      ```html
      <span>You chose #COIN_SIDE#..</span>
      ```
    </li>

    <li>TossResultText</li>

    <li className="list-none">
      ```html
      <span>#COIN_SIDE# it is..</span>
      ```
    </li>

    <li>HeadText</li>

    <li className="list-none">
      ```html
      <span>Gold</span>
      ```
    </li>

    <li>TailText</li>
    <li className="list-none">Change this based on the Coin Color in the assets of your ticket.</li>

    <li className="list-none">
      ```html
      <span>Purple</span>
      ```
    </li>
  </ul>

  <li id="media-content">Edit **MediaContent**</li>

  ![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887461/builds/coin-toss/d2yfr2tha77gnus1pxee.png)  

  <ul>
    <li>FrontImage</li>
    <li className="list-none">Coin 1 Asset with size 500x500</li>
    <li>BackImage</li>
    <li className="list-none">Coin 2 Asset with size 500x500</li>
    <li>CoinFlipLogo</li>
    <li className="list-none">Logo Asset with size 600x150, if not provided can be replaced by 1px png *(can be source from previous ticket/s)*</li>
    <li>BackgroundImage</li>
    <li className="list-none">Mobile BG Asset with size 900x700</li>
  </ul>
</ol>

### Step 3: Edit **Faqs**

There are 5 FAQs here, but only edit Faq1 & Faq3. Click the drop down icon (►) of Faqs, then go to Faq1.

1. Change the Promo Name with your Ticket's Promo Name\
   ![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887464/builds/coin-toss/rkwahubdsp6rupnuksx0.png)   
2. Proceed to Faq3. Change the Promo Date with your Ticket's Promo Date\
   ![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887463/builds/coin-toss/tcc9jddh5fmy5kemkvzk.png)
