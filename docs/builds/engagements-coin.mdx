# Coin Toss

One of the Casino Engagements is the Coin Toss. In this game, the player flips a coin and predicts which side it will land on.

<img width="100%" src={CoinTossPreview} alt="Coin Toss Page Preview" />

{/* https://promo.wv.betmgm.com/en/promo/ep/c/road-to-the-big-game-coin-toss-feb?id=26029771 */}

<br/>
To build a Coin Toss Page, you'll need to have the following sitecore files:
1. Layout
2. PAT 

## Layout
Here, you'll edit the layout of the Coin Toss. Follow the Path below to start building.

:::info[SiteCore Path]
content > Vanilla.Mobile > brand/state > Promo > Engagement > c 
:::

### Step 1: Duplicate File

<img width="100%" src={CoinTossLayout1} alt="Coin Toss Page Preview" />  
<br />

1. Right-click the file you want to duplicate.
2. Choose Copy To
3. Remove the URL of your choosen File
4. Click Copy
5. Rename the file to your URL

### Step 2: Edit **assetsandcontent**

<ol>
<li>Click the drop down icon (►) of your file, then go to assetsandcontent</li>
<li>Edit the **Toss** Section</li>
<ul>
<li>PreBetText</li>
<li className="list-none">Just edit the Gold or Purple part, depending on the color of your Coin *(as seen in the assets)*.</li>
<li className="list-none">
```html
<span className="txt-white" style="font-size: var(--site-spacer-l)">It's all in your hands </span><br />
<span>Make a bet! is it going to be</span><br/>
<span>Gold or Purple?</span>
```
</li>
<li>TossConfirmationText</li>
<li className="list-none">
```html
<span>You chose #COIN_SIDE#..</span>
```
</li>
<li>TossResultText</li>
<li className="list-none">
```html
<span>#COIN_SIDE# it is..</span>
```
</li>
<li>HeadText</li>
<li className="list-none">
```html
<span>Gold</span>
```
</li>
<li>TailText</li>
<li className="list-none">Change this based on the Coin Color in the assets of your ticket.</li>
<li className="list-none">
```html
<span>Purple</span>
```
</li>
</ul>
<li id="media-content">Edit **MediaContent**</li>
<br/>
<img width="100%" src={CoinTossLayout2} alt="Coin Toss Sitecore mediacontent" />
<br/>
<ul>
<li>FrontImage</li>
<li className="list-none">Coin 1 Asset with size 500x500</li>
<li>BackImage</li>
<li className="list-none">Coin 2 Asset with size 500x500</li>
<li>CoinFlipLogo</li>
<li className="list-none">Logo Asset with size 600x150, if not provided can be replaced by 1px png *(can be source from previous ticket/s)*</li>
<li>BackgroundImage</li>
<li className="list-none">Mobile BG Asset with size 900x700</li>
</ul>
</ol>
