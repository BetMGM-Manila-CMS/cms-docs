# Building the Voucher

After locating the voucher build on SiteCore, select a voucher build right click on it and select Copy To.![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1712820594/builds/voucher/Copy_bpgcit.jpg)

<Admonition type="note" title="Note">
  This is the icon for voucher builds <img src="https://res.cloudinary.com/dlfu36fiw/image/upload/v1712304296/builds/voucher/icon_xgza6r.jpg" className="inline m-0" />
</Admonition>

After copying the build right click on it and rename it with the URL provided by the sub leads.

# Config

Go to config.![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1712892708/builds/voucher/Screenshot_2024-04-12_113048_m1igor.jpg)

Edit the parameters :![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1712892823/builds/voucher/config_vofwtc.jpg)

cta\_login\_label: CTA Title

end\_date\_utc: Date

<Admonition type="note" title="Note">
  This is only if your build is required to change the CTA of your build, if not skip this part and go to the next part.
</Admonition>

After editing conifg click Save & Unlock > Proposed > Final![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1712821485/builds/voucher/Save_qukzsd.jpg)

# Header

After editing config go to header and add your images here by copying the URL of images under Media Library.![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1712821631/builds/voucher/header_sar1sx.jpg)

Edit the parameters below:![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1712822163/builds/voucher/Header_parameter_joae76.jpg)

Image\_mobile\_src (750x300 this is the size for mobile image)

Image\_src (955x291 this is the size for the desktop image)

#### How to get URL from Media Library

Here's the sample of location of your media library![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1712821812/builds/voucher/Media_Library_glefjt.jpg)

Locate the asset and click on the image and click open to get the URL of the image.![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1712821815/builds/voucher/Sample_image_ehr6bz.jpg)

After editing config click Save & Unlock > Proposed > Final![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1712821485/builds/voucher/Save_qukzsd.jpg)

# Description

Go to description and edit the Title and Text.![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1712822344/builds/voucher/Description_i041l9.jpg)

Edit the parameters below:![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1712822440/builds/voucher/Des_para_pks6fl.jpg)

Title: Title will be provided on copy (make sure your title is capitalized each word)

Text: Copy description and add \<p> tag on your description and close it with closing p tag \</p>

After editing config click Save & Unlock > Proposed > Final![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1712821485/builds/voucher/Save_qukzsd.jpg)

<Admonition type="note" title="Note">
  Note: Make sure to add \<br> on every \<p> tag inside text description.
</Admonition>

# Steps

Go to steps and edit step\_1.![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1712891783/builds/voucher/Steps_roila3.jpg)

Edit Title and Text.![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1712891837/builds/voucher/steps_par_znb2rq.jpg)

Title: Title will be provided on copy (make sure your title is all caps).

Text: Copy description and add \<p> tag on your description and close it with closing p tag \</p>.

Edit the parameters below:![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1712891868/builds/voucher/steps_par_j0et0e.jpg)

cta\_label: CTA Title

cta\_url: link
