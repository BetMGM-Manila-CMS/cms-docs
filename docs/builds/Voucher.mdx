# Building the Voucher

After locating the voucher build on SiteCore, select a voucher build right click on it and select Copy To.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1712820594/builds/voucher/Copy_bpgcit.jpg)

After copying the build right click on it and rename it with the URL provided by the sub leads.
