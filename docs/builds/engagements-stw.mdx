# Spin The Wheel

Spin The Wheel is an Engagement tool for players to play and win prizes.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887578/builds/stw/k3lsdnhfs2g0yprea4nv.png)

## Assets

<Admonition type="info" title="Required Assets">
  In order to properly build this component, make sure that the Box link contains a directory for the Spin the Wheel assets. These sub-directories should be included inside:

  1. **Wheel** (Wheel, Wheel Center, Ticker Color Palette)
  2. **Background** (Desktop and Mobile versions of the Background)

  ![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1727754619/builds/stw/Screenshot_2024-10-01_114834_mtkjso.png)
</Admonition>

### Design Concerns

Avoid additional details **inside** and **outside** the wheel.![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1727760481/builds/stw/extra-details_tedfpb.png)

Make sure that the top of the wheel should not be centered on the wedge

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1727761089/builds/stw/wheel-alignment_ufmwb1.png)

<Admonition type="info" title="Note">
  If you have found these issues, you can chat on the Design Concerns Teams group chat and inform the proper leads about adjusting the said concerns.
</Admonition>

## Building the Component

<Admonition type="info" title="SiteCore Path">
  **Layout:**
  Content > Vanilla.Mobile > State/Brand > Promo > BspinLayouts

  **CSS asset:**
  Content > Vanilla.Mobile > State/Brand > Promo > SpinTheWheel > PageAssets
</Admonition>

Duplicate any of the Layouts, and rename it like in the existing layouts.

You also need to create it's own CSS asset, just duplicate existing CSS and rename as such.

### Content

Fill the **PageHeaderText** with Promohub H1.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887511/builds/stw/jhvpx6qdbpknhrwfeagh.png)

Change CTA depending on your ticket if Casino/Poker.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887511/builds/stw/gh5qxcr9z4nutp3ufghn.png)

### Overlay

**PreLoginPage**: replace the img src link with the media link for Wheel with icons and the button color depending on your Brand.

Replace place this line in the html code.

```html
<img alt="" class="wheel-static-img" src="{insert img link here}" />
```

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887513/builds/stw/fri5wbtw7igwvgaweraj.png)
