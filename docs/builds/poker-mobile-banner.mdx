## Previewing

You can preview the published mobile banners with the link below. Just update the url to preview for other state and brand.
```
https://poker.nj.betmgm.com/en/mobilebanner
```

If you have applied a query filter, just add it to the original link:
```
https://poker.nj.betmgm.com/en/mobilebanner?qa=manila
```
