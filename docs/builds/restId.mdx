# Rest ID

## Creating a Rest ID

To create a Rest ID go to the Sitecore path mentioned below and duplicate an existing Rest ID Item.

<Admonition type="info" title="Sitecore Path">
  Content Editor > Marketing > US > CMS-NEW-FOLDER > state-brand > product >
  year > month
</Admonition>

Name the new created Rest ID same with the name of your ticket. Just remove unwanted characters.

<Admonition type="tip" title="Tip">
  Vivian from US provided a template folder inside the the [US Sitecore NJ Folder](http://cms-us.prod.env.works/sitecore/DirectLink.aspx?fo={64853FDD-BE01-41A1-AFBD-AC7536EC6773}\&la=), we can just `copy to` these items (except to ON) to the correct folder.

  ![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887561/builds/restId/pdegrnzz3udh8x9teyhz.png)
</Admonition>

### Inbox Terms and Conditions

If Terms and Conditions are required, always ensure that the `ShowManualTermsAndConditions` checkbox is checked. Conversely, if they are not needed, uncheck the checkbox.

Put the Terms and Conditions on the ManualTermsAndConditions Textarea.

## Commenting

Right click your Rest ID and copy both the Rest ID and the Direct Link, provide both these for your ticket comment.

![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887559/builds/restId/tzrdjmubvb0f6qjhjvcf.png)

<Admonition type="tip" title="tip">
  Embed the direct link of the Rest ID on the Rest ID itself or on a 'DL' beside it.

  You can also provide a short description for what the Rest ID is(`IOT`, `CB-Fixed No Deposit Wager`, `BS-Fixed Deposit Wager`, etc...).

  ![](https://res.cloudinary.com/dlfu36fiw/image/upload/v1709887494/builds/restId/lxefghgppnxnzz4mkebf.png)
</Admonition>
