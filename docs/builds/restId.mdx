# Rest ID

## Creating a Rest ID

To create a Rest ID go to the Sitecore path mentioned below and duplicate an existing Rest ID Item.

<Admonition type="info" title="Sitecore Path">
  Content Editor > Marketing > US > CMS-NEW-FOLDER > state-brand > product > year > month
</Admonition>

Name the new created Rest ID same with the name of your ticket. Just remove unwanted characters.

<Admonition type="tip" title="Tip">
  Vivian from US provided a template folder inside the the [US Sitecore NJ Folder](http://cms-us.prod.env.works/sitecore/DirectLink.aspx?fo={64853FDD-BE01-41A1-AFBD-AC7536EC6773}\&la=), we can just `copy to` these items (except to ON) to the correct folder.

  ![](/img/builds/restId/restid-template.png)
</Admonition>

### Grouped Rest IDs

Rest IDs needs to be grouped if a ticket is requiring more than one Rest ID for that create a folder and name it the name of your ticket. Then place the different Rest IDs inside the folder. See examples below.

## Overlay

The Overlay is a section for the Rest ID component that holds the values for a popup message.

Here is an example:

![](/img/builds/restId/restid-overlay-eg.png)

Here are the required fields for the overlay component

1. **Overlay Title** - add the overlay title
2. **Overlay Description** - add the overlay  description
3. **Overlay CTA** - add the overlay CTA. For more info refer to the CTA page.
4. **Overlay Image** - use image Overlay EDS-840x246.
5. **CTANativeLink URL** - use the URL you used for the overlay CTA
6. **CTANativeLink Link Text** - use the text you used for the overlay CTA

<Admonition type="info" title="CTA Casing">
  * Use camel casing for BetMGM
  * Use capitalized for other labels
</Admonition>

<Admonition type="info" title="CTA Styling">
  Do not add any inline style or class to the CTA.
</Admonition>

## Toaster

The Toaster is a section for the Rest ID component that holds the values for the toaster components that attracts the user's attention for notifications and messages.

Here is an example:

![](/img/builds/restId/restid-toaster-eg.png)

Here are the required fields for the toaster component:

1. Check the `Close With Timer` checkbox
2. **Toaster Title** - add the toaster title
3. **Toaster Description** - add the toaster description
4. **Toaster CTA** - add the toaster CTA. For more info refer to the CTA page.

<Admonition type="info" title="CTA Casing">
  * Use camel casing for BetMGM
  * Use capitalized for other labels
</Admonition>

<Admonition type="note" title="CTA Styling">

## Commenting

Right click your Rest ID and copy both the Rest ID and the Direct Link, provide both these for your ticket comment.

![](/img/builds/restId/restid-comment-value.png)

<Admonition type="tip" title="tip">
  Embed the direct link of the Rest ID on the Rest ID itself or on a 'DL' beside it.

  You can also provide a short description for what the Rest ID is(`IOT`, `CB-Fixed No Deposit Wager`, `BS-Fixed Deposit Wager`, etc...).

  ![](/img/builds/restId/restid-comment.png)
</Admonition>
