# Banners

## Big and Exit Banners

No need to make a Sitecore component here. Just copy the template below and update the `href` and `src` attributes with the final URL for your ticket and the scmedia URLs for the respective banners.

```html
<a href="FINAL_URL" target="_blank">
<img src="SCMEDIA_URL" border="0" /></a>
```

* **Big Banner:** `Big Banner-545x60.jpg`
* **Exit Banner:** `Exit Banner -336x280.jpg`
* **Exit Banner w/ CTA:** `Exit Banner w CTA -336x280.jpg`

To find the scmedia URLs for the above images:

1. Go to the Sitecore directory where you uploaded your ticket's creatives.
2. Select the image you need, then in the Media section, click **Open**.
   ![](/img/builds/poker-banners/scmedia-url-1.png)
3. A new tab will open. Copy the URL in the address bar, then paste it in the `src` attribute in the template.
   ![](/img/builds/poker-banners/scmedia-url-2.png)
4. Repeat for the other banner images.

## Mobile Banners

<Admonition type="info" title="Sitecore path">
  Content Editor > Vanilla.Mobile > m2.state.label > MobilePoker-v1.0 > Regional > Global > Promotions-Live

  ![](/img/builds/poker-banners/mobile-banner-path.png)
</Admonition>

If your ticket's needs require a mobile banner, here's how to create one.

1. Navigate to the Sitecore path above.
2. Create a duplicate of any component inside. Use the ticket's title as the name of the component.
3. Update the fields in the following sections:
   * **Filter**
     * **Condition:** Use your QA filter (e.g. `QueryString.Get('qa') = 'manila'`).
   * **MobileBannerData**
     * **MobileBannerImage:** `367x77.jpg`
     * **MobileBannerTitle:** Use the promotion title.
     * **MobileBannerText:** Use the promotion dates. Wrap in `<p></p>` tags and check `AlignMobileBannerTextToTheRight`.
     * **MobileBannerCTA Url:** Tournament Page smartlink (e.g., `!!M2.Poker/tournaments/TOURNEY_PAGE_URL`). See [CTA guide](https://coralracing-my.sharepoint.com/:x:/g/personal/krisha_manalo_entaingroup_com/EfvRGdc4sN5MiwcmKjBu55wBGvjTEIDYp-1XEU-iLJkGXA) for more info.
       ![](/img/builds/poker-banners/mobile-banner-data.png)
4. In the Publishing section, make sure the Publish and Unpublish dropdowns are empty (clear them if otherwise) and *Never publish* is **unchecked**.
